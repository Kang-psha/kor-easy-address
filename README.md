# kor-easy-address

사용

```javascript
var easyinput = new EasyAddress('#address-area', {
        
        apikey: "apike", //도로명 주소api => 주소검색api > apikey
       
        event :{ 
            selectevent : function(address){ //선택 되었을때의 도로명주소
                console.log(address)
            }

        }

})

```
